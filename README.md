### 🧠 CareerPath Intelligence Suite

> **AI-powered career development platform with local LLM inference, real-time AI coaching, and intelligent resume analysis.**

<br/>

**🏗 Architecture & Engineering**
- `Microservice Architecture` — Decoupled services for scalability
- `Local LLM Inference` — On-device AI via Ollama (zero API cost)
- `Server-Sent Events (SSE)` — Real-time streaming AI responses
- `MongoDB Caching Layer` — Optimized query performance
- `JWT Authentication` — Secure token-based auth flow
- `Resume PDF Parsing` — Automated document intelligence

<br/>

**🎯 Core Features**
- ATS Resume Scoring Engine
- AI-Driven Skill Gap Analysis
- Personalized Career Recommendations
- Real-Time AI Career Coach Chatbot
